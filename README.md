# TrackTalentIR – نسخهٔ جدید با رابط کاربری بهتر

اپلیکیشن فارسی استعداد‌یابی دوومیدانی با Streamlit. با وارد کردن اطلاعات و رکوردها، پیشنهاد مناسب می‌دهد، امکان آپلود ویدئو برای بررسی حرکت دارد و یک پیش‌بینی ساده از رکورد آینده ارائه می‌کند.

## اجرای سریع

۱. مخزن را دریافت کنید (در این مثال از نسخهٔ ZIP استفاده می‌کنیم):

   ```bash
   unzip TrackTalentIR_NewUI.zip
   cd TrackTalentIR_NewUI
   ```

۲. یک محیط مجازی پایتون ایجاد و فعال کنید:

   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   ```

۳. وابستگی‌ها را نصب کنید:

   ```bash
   pip install -r requirements.txt
   ```

۴. دادهٔ آزمایشی بسازید و مدل پیشنهادی را آموزش دهید:

   ```bash
   python scripts/make_demo_data.py
   python scripts/train_text.py
   ```

۵. اپلیکیشن را اجرا کنید:

   ```bash
   streamlit run streamlit_app.py
   ```

در مرورگر، صفحهٔ اصلی رضایت‌نامه را مشاهده می‌کنید. پس از تأیید رضایت، می‌توانید
از منوی سمت راست به فازهای دیگر بروید.

## ساختار مخزن

- `streamlit_app.py` – فایل اصلی رابط کاربری با منوی چندبخشی.
- `app/engines/` – پیاده‌سازی سادهٔ موتورهای سه‌گانه (توصیه‌گر، تحلیل حرکت، پیش‌بینی رکورد).
- `data/` – محل ذخیرهٔ داده‌ها. فایل `talent_train.csv` پس از اجرای اسکریپت داده‌سازی اینجا ساخته می‌شود.
- `models/` – مدل‌های آموزش‌دیده در این پوشه ذخیره می‌شوند.
- `scripts/` – اسکریپت‌های خط فرمان برای ساخت داده و آموزش مدل.
