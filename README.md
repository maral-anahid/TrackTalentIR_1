# TrackTalentIR – نسخهٔ جدید با رابط کاربری بهتر

TrackTalentIR یک اپلیکیشن سه‌فازه برای استعداد‌یابی ورزش دوومیدانی است. این نسخه یک رابط کاربری
جدید و ساده با Streamlit فراهم می‌کند که تمام بخش‌ها را در یک صفحهٔ اصلی با منوی
سمت راست ارائه می‌دهد. شما می‌توانید فرم استعداد‌یابی را پر کنید، ویدئوهای کوتاه
خود را برای تحلیل حرکت بارگذاری کنید و پیش‌بینی رکورد آیندهٔ خود را مشاهده کنید.

## اجرای سریع

۱. مخزن را دریافت کنید (در این مثال از نسخهٔ ZIP استفاده می‌کنیم):

   ```bash
   unzip TrackTalentIR_NewUI.zip
   cd TrackTalentIR_NewUI
   ```

۲. یک محیط مجازی پایتون ایجاد و فعال کنید:

   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   ```

۳. وابستگی‌ها را نصب کنید:

   ```bash
   pip install -r requirements.txt
   ```

۴. دادهٔ آزمایشی بسازید و مدل پیشنهادی را آموزش دهید:

   ```bash
   python scripts/make_demo_data.py
   python scripts/train_text.py
   ```

۵. اپلیکیشن را اجرا کنید:

   ```bash
   streamlit run streamlit_app.py
   ```

در مرورگر، صفحهٔ اصلی رضایت‌نامه را مشاهده می‌کنید. پس از تأیید رضایت، می‌توانید
از منوی سمت راست به فازهای دیگر بروید.

## ساختار مخزن

- `streamlit_app.py` – فایل اصلی رابط کاربری با منوی چندبخشی.
- `app/engines/` – پیاده‌سازی سادهٔ موتورهای سه‌گانه (توصیه‌گر، تحلیل حرکت، پیش‌بینی رکورد).
- `data/` – محل ذخیرهٔ داده‌ها. فایل `talent_train.csv` پس از اجرای اسکریپت داده‌سازی اینجا ساخته می‌شود.
- `models/` – مدل‌های آموزش‌دیده در این پوشه ذخیره می‌شوند.
- `scripts/` – اسکریپت‌های خط فرمان برای ساخت داده و آموزش مدل.

## ویژگی‌ها

1. **توصیه‌گر توضیح‌پذیر:** یک مدل سادهٔ Logistic Regression روی داده‌های
   فرمی، برای پیشنهاد ۳ گروه مادهٔ دوومیدانی به کاربر.
2. **تحلیل ویدئو:** در این نسخه یک تحلیل آزمایشی انجام می‌شود و امتیازی
   تصادفی به کاربر نمایش داده می‌شود. می‌توانید این قسمت را با یک مدل
   واقعی (مثل MediaPipe) جایگزین کنید.
3. **پیش‌بینی رکورد:** یک پیش‌بینی آزمایشی بر اساس میانگین وزنی رکوردهای اخیر
   و کمی تغییر تصادفی انجام می‌شود.

این نسخهٔ ساده جهت نمایش رابط کاربری و نحوهٔ ارتباط بخش‌هاست. برای استفادهٔ واقعی،
باید از داده‌ها و مدل‌های معتبرتر استفاده و موتورهای تحلیل را توسعه دهید.
